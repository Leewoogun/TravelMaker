## 🗼 TravelMaker 소개
- TravelMaker는 사용자가 여행을 떠났을 때 그 여행지의 **로컬 현지인과의 만남**을 쉽게 할 수 있도록 도와주고 그 **만남의 추억을 편하게 기록**을 할 수 있게 도와주는 서비스 입니다.
